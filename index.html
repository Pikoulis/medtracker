One-time setup in Supabase

Create a free project, then open SQL Editor and run this once. Copy the Project URL and anon key from Settings → API. 

create table medications (
  id text primary key,
  name text,
  qty numeric,
  daily numeric,
  unit text,
  "start" text,
  remind integer
);
alter table medications enable row level security;
create policy "anon access" on medications
  for all to anon using (true) with check (true);

Anyone with your URL and anon key can read this table, so don't share them or commit them anywhere public. For multiple separate people on the same project, each should use their own Supabase project instead.
